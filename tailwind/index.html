Guía de clases de Tailwind CSS

🎨 Colores

bg-red-500 → Fondo rojo
bg-green-500 → Fondo verde
bg-blue-500 → Fondo azul
text-red-500 → Texto rojo
text-green-500 → Texto verde
text-blue-500 → Texto azul

🔤 Tipografía

text-xs → Texto muy pequeño

text-base → Texto base

text-xl font-bold → Texto grande y en negrita

italic → Texto en cursiva

underline → Texto subrayado

line-through → Texto tachado

uppercase → Mayúsculas

📏 Espaciado (margin/padding)

m-4 → Margin externo 1rem
p-4 → Padding interno 1rem
px-6 py-2 → Padding horizontal 1.5rem, vertical 0.5rem

🔲 Bordes

🌑 Sombras

shadow-sm → Sombra pequeña
shadow-lg → Sombra grande
shadow-2xl → Sombra muy grande

🌈 Degradados

bg-gradient-to-r → Izquierda a derecha
bg-gradient-to-b → De arriba hacia abajo
bg-gradient-to-tr → Diagonal arriba derecha
bg-gradient-to-l → De derecha a izquierda
